### 4.3 Calculation Logic

#### Step 1: Fuel Required

```
Fuel Needed = Distance / Mileage
```

#### Step 2: Total Fuel Cost

```
Total Cost = Fuel Needed √ó Fuel Price
```

#### Step 3: Split Count

```
If Include Owner = true:
    Split Count = People
Else:
    Split Count = People - 1
```

#### Step 4: Cost Per Person

```
Each Pays = Total Cost / Split Count
```

#### Step 5: Apply Rounding

* Exact ‚Üí no change
* ‚Çπ1 ‚Üí round(amount/1)*1
* ‚Çπ5 ‚Üí round(amount/5)*5
* ‚Çπ10 ‚Üí round(amount/10)*10
